- AI是利润中心吗？→ 是，BIP是对外卖的产品

| **业务方向** | 工程机械智能制造、工业互联网（树根互联）、无人驾驶矿车 |
| **AI定位** | 制造型企业中AI落地最激进的之一，有"灯塔工厂" |
| **目标岗位** | 智能制造系统架构师 / AI产品经理 / 知识图谱算法工程师 |
| **匹配度分析** | 制造场景经验直接可用 / 数据基建比理想汽车更成熟（自有工厂） |

**五问检验预判：**
- AI是利润中心吗？→ 部分是（树根互联对外输出），部分是内部提效

**适合你的点：** 如果三一的数据治理确实比理想成熟（有"灯塔工厂"可能是正面信号），你能专注在AI应用层；但不能默认"自有工厂=数据好"
