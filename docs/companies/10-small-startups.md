# 海淀/昌平 小型工业AI创业公司筛选

> 调研日期：2026-06-08
> 筛选标准：<100人、工业AI/制造智能化方向、已融资、北京海淀/昌平

---

## 筛选逻辑

你提到过"小公司也是一种选择"——小团队意味着：
- 人际关系简单，无大公司政治
- 个人影响力大，能直接推动决策
- 可能拿到期权/股份
- 成长速度快（但稳定性低）

**核心筛选条件：**
1. 团队<100人（优选<50人）
2. 方向与"工业AI落地"相关（预测维护/质量AI/供应链/制造数字化）
3. 至少完成A轮融资（排除纯PPT公司）
4. 北京办公（优选海淀）
5. 你的能力能发挥核心价值（不只是写代码）

#### 为什么高度匹配

| 你的能力 | 在天泽智云的映射 |
|---------|-----------------|
| 供应链质量预警 | **直接对口** — 他们核心就是设备故障预测 |

| 数据闭环建设 | ✓ 工业数据采集→建模→反馈是他们的产品逻辑 |

#### 可能的角色

- **AI解决方案负责人** — 负责将PHM技术落地到新行业/新客户
- **产品总监** — 定义PHM产品的标准化和行业化

#### 薪资预估

| 级别 | 预估 | 备注 |

| 总监级 | 50-70万 + 期权 | 小公司现金可能略低，期权补偿 |

## 我的判断

> **天泽智云是所有小公司中最匹配的 — 几乎是为你量身定制的公司。**

理由：
- 预测性维护/PHM = 你在理想做过的设备异常预警的升级版
- ~50人团队 = 人际简单、影响力大
- 海淀知春路 = 通勤友好
- "专精特新" = 有政府背书，不是PPT公司
- 他们最缺的 = "能把PHM技术卖给客户+量化价值"的人 = 你

**但核心风险：**
- 小公司稳定性不如大公司/国企
- 可能需要出差到客户工厂
- 50人团队 = 资源有限，一人多角色
- 对宝妈来说，"稳定性vs个人发展"的取舍更尖锐

**建议策略：**
- 天泽智云/寄云科技作为"高匹配备选"，与用友/第四范式并行推进
- 面试时重点确认：出差频率、弹性工作、团队分工
- 如果能拿到"不常出差"的角色（如产品定义/方法论），是非常好的选择
