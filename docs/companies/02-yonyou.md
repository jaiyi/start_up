## 目标岗位分析

### 岗位A：AI产品经理（智能制造方向）

#### 核心职责

1. **AI场景规划** — 挖掘生产制造、供应链、质量管理环节的AI落地场景（智能排产、预测性维护、视觉质检、工艺优化）
2. **产品全生命周期** — 需求分析、原型设计、PRD撰写，协同研发完成模型训练到应用上线
3. **大模型应用落地** — 推动LLM在工业场景的Agent化应用
4. **数据资产运营** — 定义工业数据标准，构建工业知识库
5. **市场与竞品分析** — 洞察工业AI行业趋势

#### 任职要求 vs 你的匹配度

| 要求 | 具体内容 | 你的匹配度 |
|------|---------|-----------|
| 学历 | 本科+，计算机/自动化/工业工程 | ✓ GT EE/CS硕士 |
| 经验 | 3-5年+产品经验，有ERP/MES/工业互联网经验优先 | ✓ 6年工业智能，有MES/产线对接经验 |
| AI能力 | 熟悉LLM技术原理，有AI项目落地经验 | ✓ 有AI落地实战 |
| 业务理解 | 深刻理解制造业生产流程 | ✓ 核心优势 |
| 软技能 | 跨部门沟通、项目推动 | ✓ |

**匹配度：85%** — 这个岗位几乎是为你量身定做的

### 岗位B：解决方案架构师（智能制造方向）

#### 核心职责

1. **顶层架构设计** — BIP制造整体方案，确保AI与ERP/MES/PLM深度集成
2. **客户需求转化** — 深入大型制造企业，输出数字化转型方案
3. **技术方案落地** — 技术选型、架构评审、解决技术瓶颈
4. **生态协同** — 协同内部研发/售前/交付 + 外部AI合作伙伴
5. **标杆案例打造** — 沉淀行业解决方案模板

#### 任职要求 vs 你的匹配度

| 要求 | 具体内容 | 你的匹配度 |
|------|---------|-----------|
| 学历 | 本科+，计算机/软件工程/自动化 | ✓ |
| 经验 | 5-8年大型制造业信息化/数字化项目 | ✓ 6年 |
| 技术深度 | 精通企业级架构（微服务、云原生） | △ 了解但不精通 |
| 行业洞察 | 离散/流程制造深刻理解 | ✓ 汽车制造（离散） |
| 沟通表达 | 方案宣讲、客户高层沟通 | △ 内部经验为主 |

**匹配度：70%** — 技术深度要求更高，更偏架构师

### 推荐岗位选择

**AI产品经理 > 解决方案架构师**

理由：
- PM岗匹配度更高（85% vs 70%）
- PM岗在北京总部，出差少
- PM岗更偏"定义产品"方向 = 正好补你"产品定义力"的短板
- 架构师岗技术深度要求更高，且可能需要出差到客户现场

## 综合评估

### 适合你的理由

1. **通勤零成本** — 永丰产业园，步行/骑车可达，对宝妈是巨大优势
2. **产品岗不出差** — 北京总部做产品规划，偶尔调研，不驻场
3. **正好补"产品定义力"短板** — PM岗就是让你定义AI产品
4. **AI是公司级战略** — 不是边缘部门，是CEO亲推的核心方向
5. **制造业基因深厚** — 30年服务制造企业，团队有大量行业专家
6. **信创替代SAP/Oracle** — 大量央国企制造客户正在迁移，需求爆发期
7. **稳定性高** — 老牌上市公司，不会突然倒闭

### 风险与顾虑

1. **薪资可能偏低** — 如果给不到总监级，可能年薪降到50-60万
2. **大公司病** — 流程繁琐、决策链长、内部协调成本高
3. **技术转型阵痛** — 从传统ERP转型AI，内部可能有新旧思维冲突
4. **天花板问题** — 大公司内上升空间有限，你可能又回到"连长"位置
5. **不够"创业感"** — 如果你追求的是小团队的灵活和快反馈，这里可能太大了
