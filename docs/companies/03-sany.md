
## 目标岗位分析

### 岗位：智能制造/AI 产品经理

#### 核心职责

1. **场景挖掘** — 深入灯塔工厂、智慧矿山、智慧港口，发现业务痛点
2. **需求转化** — 将排产优化、质量预警、设备预测维护等问题转化为产品需求
3. **AI落地协同** — 协同算法团队推动视觉AI、大模型、运筹优化在制造流程中落地
4. **产品全生命周期** — 需求分析→原型设计→推进开发→上线评估效果

## 综合评估

### 适合你的理由

1. **制造业场景极其丰富** — 灯塔工厂、矿山、港口，AI落地场景多
2. **高管真的在推** — 数字化是生死存亡级的战略
3. **你的经验直接可用** — 质量预警、设备预测维护都是他们的需求
4. **履历含金量高** — "三一灯塔工厂"的项目经验在市场上很值钱

### ⚠️ 核心风险（对你而言可能是dealbreaker）

1. **工作强度极高** — 军事化管理 + "嫉慢如仇"文化，对新晋宝妈极不友好
2. **出差不可避免** — 产品岗需要去工厂/矿山一线
3. **薪资可能降幅大** — 制造业天花板偏低，60-80万需很高级别
4. **管理风格传统** — 层级分明，与你追求的"小团队、人际简单"相悖
5. **场景迁移成本** — 工程机械 vs 汽车制造，业务逻辑差异不小

## 我的判断

> **三一重工对你的情况来说，风险收益比不合理。**

理由：
- 工作强度和管理风格与你"兼顾家庭、人际简单"的诉求严重冲突
- 薪资可能需要降幅
- 场景虽好，但你在理想的6年经验已经足够证明制造业场景能力
- 更适合"不需要兼顾家庭、处于职业冲刺期"的候选人

**如果一定要考虑三一生态，建议看树根互联** — 它是三一孵化但独立运营的工业互联网平台，对外卖AI能力（利润中心），文化可能比三一本体更互联网化。但需要确认其北京办公地点。
