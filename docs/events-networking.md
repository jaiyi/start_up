## Networking策略建议

### 核心原则

> 参加一次30-50人的高质量沙龙 > 参加一次3000人的大型展会

**中国行业networking文化的关键：**
- 活动现场主动交换微信 → 进入3-5个相关群
- 群内持续输出有价值的观点 → 建立个人品牌
- 从"旁听者"到"分享者"的转变 → 被邀请做嘉宾

### 推荐路径

1. **第一步（本月）：** 关注所有公众号 + 注册活动行
2. **第二步（6月）：** 参加AIII 2026（6/26-28）或智源大会（6/12-13）
3. **第三步（7-8月）：** 参加1-2次小型沙龙，开始交换联系方式
4. **第四步（持续）：** 每月至少参加一次行业活动，进入核心圈子
5. **长期：** 申请在某个沙龙做10-15分钟分享（如"汽车制造AI落地的5个坑"），快速建立行业认知

### 你的独特分享话题（可用于沙龙演讲）

- "EVI方法论：如何量化AI在制造业的经济价值"
- "从理想汽车到行业通用：制造业AI落地的3个结构性问题"
- "供应链质量预警：从0到3亿计价收益的实战复盘"

这些话题在工业AI圈子里有很高的稀缺性和吸引力。
